# **MakeUrPC**

**MakeUrPC** — это веб-сервис для онлайн-конфигурации персональных компьютеров на базе Spring Boot. Сервис позволяет пользователям собирать ПК из совместимых комплектующих. Приложение предоставляет полноценное REST API для управления комплектующими, конфигурациями и проверки совместимости компонентов.

---

## Основные функции

1. **Функционал для пользователя**:

    - Подбор комплектующих (процессор, видеокарта, ОЗУ, материнская плата, БП, корпус, накопители).
    - Автоматическая проверка совместимости компонентов в реальном времени.
    - Сравнение конфигураций.

2. **Базовый функционал API**:

    - GET-запросы с возвращением данных в виде JSON (списки комплектующих, конфигурации).
    - POST-запросы для создания новых конфигураций и добавления комплектующих.
    - PUT-запросы для обновления существующих конфигураций.
    - DELETE-запросы для удаления конфигураций и компонентов.

3. **Дополнительные возможности**:

    - Фильтрация комплектующих по параметрам (сокет, TDP, форм-фактор, объем памяти).
    - Валидация совместимости на уровне бизнес-логики (бэкенд).

---

## Технологии

- **Язык программирования**: Java 25
- **Фреймворк**: Spring Boot 4.x
- **База данных**: PostgreSQL
- **Сборка**: Maven
- **Дополнительно**: Spring Data JPA, Spring Validation, Lombok

---

### Требования

- **Установленная Java 25 или выше**.
- **Установленный Maven**.
- **Установленная PostgreSQL 14+**.
- **Доступ к интернету** (для загрузки зависимостей).

---

### **Функционал**

- Запуск и работа REST API для управления/создания конфигураций ПК.
- Работа с базой данных (**PostgreSQL**) для хранения комплектующих и конфигураций.
- Валидация совместимости компонентов.
